	- Stoyboard viewing implemented

	- Tram wait times updated to match real-time wait time until arrival

Build instructions
	- In Android Studio 
		- File -> New -> Project From Version Control
		- Copy and Paste this repo URL into URL
		- Select Clone
		- Once cloned, build project

Installation
	- Download .apk file from releases
	- Copy file to android phone
	- On phone open and install .apk file
